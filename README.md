# 🎼 DP (Duckzil notePad)
<hr/>

## 🔊 Hot to start This Project?
1. download ZIP
2. type `npm init`
3. before running this app u have to upload img to the `/public` folder
4. [http://localhost:3000](http://localhost:3000)

or just visit URL (주소 추우 업로드 예쩡)
<hr/>
